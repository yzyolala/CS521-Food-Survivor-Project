We have used Bootstrap 5 for User Interface while refrencing online available free templates as this project is focused on the backend rather than a fronted. Talking about backend we have used Nodejs at core to develop entire Application and Express, Express-handlebars to handle page routings and core app functions. We have used MongoDB to handle the database for the App as well as store all the data. To securely store user data we have used Bcrypt and Bcryptjs to hash the user passwords and then store it in our database, We have even used xss to secure application from vulnerablities over the web.
